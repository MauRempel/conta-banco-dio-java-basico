# 💻 Conta Terminal - Desafio Java Básico (DIO)

Este projeto foi desenvolvido como parte do **desafio da trilha de Java Básico da DIO (Digital Innovation One)**.

O objetivo é praticar conceitos fundamentais da linguagem Java, como entrada de dados pelo terminal, uso de variáveis, manipulação de strings e estrutura básica de um programa Java.

---

## 📌 Descrição do Projeto

O programa simula a criação de uma conta bancária via terminal.

O usuário informa os dados da conta e, ao final, recebe uma mensagem de confirmação contendo todas as informações fornecidas.

### Informações solicitadas ao usuário

- Número da conta
- Número da agência
- Nome do cliente
- Saldo da conta

Após a entrada dos dados, o sistema retorna uma mensagem personalizada confirmando a criação da conta.

---

## 🧠 Conceitos praticados

Neste desafio foram utilizados conceitos fundamentais de Java:

- Estrutura de um programa Java (`public class` e `main`)
- Variáveis e tipos de dados (`int`, `double`, `String`)
- Entrada de dados com `Scanner`
- Interação com o usuário via terminal
- Concatenação de strings
- Organização básica de código

---

## 🛠️ Tecnologias utilizadas

- **Java**
- **Scanner (java.util.Scanner)**
- Terminal / Console
